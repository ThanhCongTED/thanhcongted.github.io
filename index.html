Install Firmware

Open USB Console

Installation Notes:

  • Switch ESP32 to Bootloader mode:
    • Keep FLASH button pressed while pressing RESET button
    • If no FLASH button: pull GPIO0 low
  • Manual reset might be required after installation
  • Not all web browsers support USB access (use Chrome/Edge/Firefox)
  • Check the documentation for additional information

SD Card Setup:

  • SD card can be setup automatically after firmware installation
  • SD card must be FAT formatted (default on new SD cards)
  • Alternative: Manual SD card setup (see documentation)
