How to flash:

  1. Connect ESP board to computer via USB
  2. Click "Connect ESP Device" and select the serial port
  3. Put ESP into bootloader mode:
    • ESP32: Hold BOOT button, press RESET, release RESET, then release BOOT
    • ESP8266: Hold FLASH/GPIO0, press RESET, release RESET, then release FLASH
  4. Select firmware file and configuration options
  5. Click "Flash Firmware"
